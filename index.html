Comment investir ?

  • Connectez-vous sur app.kalmydas.com avec MetaMask ou Google
  • Achetez un Access Pass (abonnement requis pour investir)
  • Déposez du USDC dans la pool de votre choix
  • Vos fonds sont gérés par le robot — retirez 24h/24 quand vous voulez
  • Les gains sont distribués automatiquement via les smart contracts

Token KAL

Le token utilitaire de l'écosystème KalMydas. Il sert à la gouvernance, au staking de liquidité, et comme bonus de récompense pour les déposants.

Bonding Curve
Le prix du KAL augmente automatiquement avec la demande. Plus il y a d'acheteurs, plus le prix monte.
Récompenses LP
Fournissez de la liquidité USDC/KAL sur KalSwap et gagnez des KAL en staking.
Airdrop
Les premiers utilisateurs reçoivent des KAL gratuits avec un vesting progressif (6 tranches).
Buyback & Burn
Une partie des frais de performance est utilisée pour racheter et brûler du KAL, réduisant l'offre.

Access Pass

L'Access Pass est un NFT d'abonnement qui donne accès aux pools d'investissement. Sans Access Pass actif, il est impossible de déposer dans une pool.

  • Prix : 1 USDC par jour (payé en USDC)
  • Durée flexible : 30, 90, 180 ou 365 jours
  • Renouvelable à tout moment avant ou après expiration
  • NFT ERC-721 : vous en êtes le propriétaire sur la blockchain

Frais & High-Water Mark

  • Frais de gestion : ~1.2% à ~6% par an selon la pool (prélevés graduellement)
  • Frais de performance : 10% à 20% sur les gains uniquement
  • High-Water Mark : Les frais de performance ne sont prélevés que quand la pool dépasse son précédent record. Pas de double facturation sur les mêmes gains.
  • Pas de frais d'entrée/sortie : Déposez et retirez sans aucun frais supplémentaire

Sécurité

  • Réseau Base L2 : Layer 2 d'Ethereum (Coinbase) — frais très bas, sécurité Ethereum
  • Circuit Breakers : Le bot se désactive automatiquement si le gain/perte journalier dépasse un seuil

  • Contrats non-upgradable : Le code des contrats ne peut pas être modifié après déploiement

FAQ

D'où viennent les rendements ? ▼
Les rendements proviennent exclusivement du trading algorithmique sur l'or (XAUUSD) via MetaTrader 4. Chaque pool est connectée à un robot de trading réel. Il n'y a pas de mécanisme Ponzi ou de rendement artificiel.
Puis-je retirer mes fonds à tout moment ? ▼
Oui. Il n'y a pas de période de lock-up. Vous pouvez retirer vos fonds 24h/24, 7j/7. Le retrait est instantané via le smart contract.
Qu'est-ce que l'Access Pass ? ▼
L'Access Pass est un NFT d'abonnement à 1 USDC/jour. Il est nécessaire pour déposer dans les pools. Sans Access Pass actif, vous pouvez consulter les performances mais pas investir.
Comment fonctionne le High-Water Mark ? ▼
Le High-Water Mark garantit que les frais de performance ne sont prélevés que sur les nouveaux gains. Si la pool perd de la valeur puis remonte, les frais ne sont prélevés qu'une fois le précédent record dépassé.
Est-ce que je peux perdre mon capital ? ▼
Oui, comme tout investissement, il y a un risque de perte. Les robots tradent sur les marchés réels. Les pools prudentes (Horizon) ont un risque limité, tandis que les pools offensives (Orion) peuvent subir des pertes importantes.
Sur quel réseau blockchain fonctionne KalMydas ? ▼
KalMydas fonctionne sur Base, le Layer 2 d'Ethereum développé par Coinbase. Actuellement en testnet (Base Sepolia), la migration vers le mainnet est prévue prochainement.
